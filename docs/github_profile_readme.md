## 🛠 Featured Project: HoverPay 📱⚡

**HoverPay** is an AI-powered ambient payment ecosystem that reimagines modern UPI checkouts. Instead of manual QR scanning and PIN-typing, it turns digital transactions into zero-friction gesture releases using Bluetooth Low Energy (BLE) beacons, face biometric scanning, and real-time desktop-mobile state pairing.

🛸 **[Explore HoverPay Repository](https://github.com/nah70909-glitch/hoverpay)** | 🔗 **[Launch Live Demo](https://courageous-bublanina-8ae4b7.netlify.app/)**

### Key Engineering Accomplishments inside HoverPay:
* 📡 **Real-Time Pairing Synchronization**: Built a pure client-side device-to-device pairing sync over Server-Sent Events (SSE), allowing physical phones to control desktop session checkouts in real-time.
* 🛡 **High-Fidelity Biometric Simulator**: Crafted an iris and pupil depth biometric sweep framework utilizing real-time device webcam overlays and spatial tracking.
* 💬 **Dual-Gateway SMS Delivery**: Designed a robust REST SMS delivery helper with Twilio basic authentication and Textbelt API dispatch fail-safes.
* 🎨 **Bespoke Visual System**: Designed a gorgeous glassmorphic interface featuring notch configuration panels, dynamic merchants radar feeds, and sub-0.8s responsive invoice transitions.
