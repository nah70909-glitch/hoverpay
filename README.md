# HoverPay

HoverPay is a futuristic gesture-assisted UPI payment experience prototype built to explore touchless fintech interactions and modern payment workflows.

## Features

* Gesture-assisted payment verification
* Modern fintech dashboard
* Merchant payment flow
* Mobile-first responsive UI
* Realistic UPI interaction design
* Firebase integration
* Smooth animations and transitions

## Tech Stack

* Next.js
* React
* Tailwind CSS
* TypeScript
* Firebase
* Framer Motion
